What is LAMP?

LAMP, commonly referred to as the LAMP Stack, is a widely used open source web platform. The acronym LAMP stands for Linux Apache MySQL and PHP/Python/Perl. It is estimated that the LAMP stack accounts for upwards of 80% of the sites connected to the web. LAMP is my secondary development stack

What is MEAN?

MEAN is a relatively new open source software stack built in JavaScript. MEAN is the acronym used to describe the technologies; MongoDB, Express.js, Angular.js and Node.js. Together these technologies become a powerful web application software bundle. I use MEAN as my primary development stack.

What is OOP?

OOP is short for Object Oriented Programming. This is a way of modeling your code to have properties and behaviors similar to real world objects. Using OOP can make code more modular and secure.

Technologies

  • Unified Modeling Language
  • Design Patterns
  • Inheritance

What is a CMS?

CMS is an acronym for Content Management System. This is a system that allows for collaboration, security and redundacy with code. Using a CMS is vital on large projects. It ensures accountability and can allow for several things to be worked on at the same time.
